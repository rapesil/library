# Library API REST for Practice Testing

This project is in Progress. Some bugs exist on purpose, but some still need to be discovered.

## About the project

To run locally you need install:

* Java 17

Then run:

 ```
 ./gradlew bootRun
 ```

The project will run in port 8080.
